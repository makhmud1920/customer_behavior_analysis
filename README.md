# customer_behavior_analysis
Analyzed customer purchasing patterns using Python, PostgreSQL, and Power BI. The project involves data cleaning, storage, and visualization to extract actionable insights.

## Features
- Data cleaning and preprocessing
- Storing data in PostgreSQL database
- Creating interactive visualizations in Power BI
- Analyzing customer behavior and trends

## Technologies
- **Python** (Pandas)
- **PostgreSQL**
- **Power BI**

## Notes
This project was completed with the help of ChatGPT and online tutorial videos.
